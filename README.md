# Bingo grego – Xerador de cartóns e bombo interactivo

Autor: Alejandro Morón Bandín  
Ano: 2026  

## Descrición

Aplicación desenvolvida en Streamlit para a práctica do alfabeto grego en 1º de Bacharelato.

Inclúe:
- Bombo interactivo de letras maiúsculas e minúsculas.
- Xerador automático de series de 6 cartóns en PDF (formato A4 apaisado).
- Deseño configurable dende a interface.

## Finalidade didáctica

Este recurso emprégase para:

- Reforzar o recoñecemento visual das letras gregas.
- Traballar a diferenciación entre maiúsculas e minúsculas.
- Introducir dinámicas de gamificación na aula.
- Fomentar a participación activa do alumnado.

## Tecnoloxías empregadas

- Python
- Streamlit
- ReportLab
